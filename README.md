# 🌸 Iris Data Visualization Project

## 📌 Overview

This project is part of my internship Task 1, where I explored and visualized the famous **Iris dataset** using Python.

## 🎯 Objectives

* Understand dataset structure
* Perform basic data analysis
* Create visualizations to extract insights

## 🛠️ Tools & Libraries

* Python
* Pandas
* Matplotlib
* Seaborn

## 📊 Visualizations

The following charts were created:

1. **Scatter Plot**

   * Shows relationship between sepal length and width
   * Helps differentiate species

2. **Histogram**

   * Displays distribution of petal length

3. **Box Plot**

   * Detects outliers and spread of data
